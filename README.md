# Olist E-Commerce Data Analysis

## Project Overview
This project analyzes the Olist e-commerce dataset to understand order trends, delivery performance, and customer behavior.

## Objectives
- Analyze order status distribution
- Study yearly and monthly trends
- Evaluate delivery performance
- Identify delays and outliers

## Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## Key Insights
- ~96% of orders are successfully delivered
- Orders increased significantly from 2017 to 2018
- Deliveries are on average 12 days earlier than expected
- Peak sales observed between May–August
- Drop in orders during September

## Conclusion
The analysis shows strong logistics performance with early deliveries, but seasonal demand variations impact order trends.
